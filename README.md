# 🧪 React Automated Testing & TypeScript

Este repositório contém o projeto desenvolvido durante o curso de testes automatizados, focado em garantir a qualidade e a segurança de aplicações React utilizando **TypeScript**, **Jest** e **React Testing Library**.

## 🚀 O que foi desenvolvido

O objetivo principal foi aprender a validar o comportamento de componentes e hooks, garantindo que a aplicação funcione exatamente como esperado através de uma suíte de testes robusta.

### Principais aprendizados:
*   **Jest & React Testing Library:** Configuração e execução de testes de unidade e integração.
*   **TypeScript em Testes:** Tipagem rigorosa para evitar erros de desenvolvimento e aumentar a clareza.
*   **Padrão AAA (Arrange, Act, Assert):** Estruturação de testes para melhor legibilidade e manutenção.
*   **TDD (Test Driven Development):** Desenvolvimento guiado por testes para um código mais limpo.
*   **Mocks & Spies:** Simulação de dependências e monitoramento de chamadas de funções.

## 🛠️ Tecnologias
*   [React](https://reactjs.org)
*   [TypeScript](https://www.typescriptlang.org)
*   [Jest](https://jestjs.io)
*   [React Testing Library](https://testing-library.com)

## 🏁 Como rodar os testes

1.  **Clone o repositório:**
    ```bash
    git clone 
    ```

2.  **Instale as dependências:**
    ```bash
    npm install
    ```

3.  **Execute a suíte de testes:**
    ```bash
    npm test
    ```
